# 🧘‍♂️ Breath of the Bald (HarAIkiri)

[![Flutter](https://img.shields.io/badge/Flutter-3.x-02569B?logo=flutter)](https://flutter.dev)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**PL:** Profesjonalna aplikacja do medytacji i treningu oddechowego, stworzona aby pomóc Ci odzyskać spokój w chaotycznym świecie.  
**ENG:** A professional mindfulness and breathing exercise application designed to help you find peace in a chaotic world.

---

## 🇵🇱 Język Polski

### 🌟 Kluczowe Funkcje
- **Trening Oddechowy:** Zróżnicowane poziomy trudności (od początkującego do mistrza zen), dostosowane do Twojego tempa.
- **Inteligentny Harmonogram:** Planuj sesje i otrzymuj przypomnienia bezpośrednio w swoim kalendarzu systemowym (Google Calendar, Outlook itp.).
- **Śledzenie Postępów:** Wizualizacja Twojej drogi do spokoju za pomocą interaktywnych wykresów i statystyk.
- **Głęboka Imersja:** Minimalistyczny interfejs z płynnymi animacjami i relaksującymi dźwiękami.
- **Integracja ze Zdrowiem:** Możliwość synchronizacji danych z systemowymi aplikacjami zdrowotnymi.

### 🛠️ Stos Technologiczny
- **Framework:** Flutter (Dart)
- **Zarządzanie stanem:** Riverpod (skalowalna i testowalna architektura)
- **Baza danych:** Drift (SQLite) – szybki i bezpieczny zapis lokalny
- **Animacje:** Flutter Animate
- **Kalendarz:** Add2Calendar (automatyczna integracja z systemem)
- **Audio:** Just Audio (wysoka jakość dźwięku)

### 🚀 Instalacja
1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/lysyokrutnik/BreathOfTheBald_v1.git
   ```
2. Pobierz zależności:
   ```bash
   flutter pub get
   ```
3. Uruchom aplikację:
   ```bash
   flutter run --release
   ```

## 🇺🇸 English

### 🌟 Key Features
- **Breathing Training:** Diverse difficulty levels (from beginner to Zen master), adapted to your rhythm.
- **Smart Scheduler:** Plan your sessions and get reminders directly in your system calendar (Google Calendar, Outlook, etc.).
- **Progress Tracking:** Visualize your journey to tranquility with interactive charts and statistics.
- **Deep Immersion:** Minimalist interface with smooth animations and relaxing sounds.
- **Health Integration:** Ability to sync data with system health applications.

### 🛠️ Tech Stack
- **Framework:** Flutter (Dart)
- **State Management:** Riverpod (scalable and testable architecture)
- **Database:** Drift (SQLite) – fast and secure local storage
- **Animations:** Flutter Animate
- **Calendar:** Add2Calendar (automatic system integration)
- **Audio:** Just Audio (high-fidelity sound)

### 🚀 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/lysyokrutnik/BreathOfTheBald_v1.git
   ```
2. Get dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run --release
   ```
